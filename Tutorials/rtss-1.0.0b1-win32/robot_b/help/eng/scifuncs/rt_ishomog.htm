rt_ishomog - test if argument is a homogeneous transformation

Calling Sequence

r = rt_ishomog(x)

Parameters

  • x : any Scilab object.
  • r : boolean. It is the test result.

Description

This function returns true (%T) if x is a 4-by-4 matrix, else returns false (%F).
