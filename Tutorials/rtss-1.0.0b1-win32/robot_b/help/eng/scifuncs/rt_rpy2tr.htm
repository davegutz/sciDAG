rt_rpy2tr - Roll/pitch/yaw angles to homogeneous transform

Calling Sequence

T = rt_rpy2tr(r, p, y)
T = rt_rpy2tr(RPY)

Parameters

  • r : scalar. The roll angle in radians.
  • p : scalar. The pitch angle in radians.
  • y : scalar. The yaw angle in radians.
  • RPY : 3-element row vector. This vector is formed by stacking the angles roll, pitch and yaw as follows RPY = [roll, pitch, yaw].
  • T : 4-by-4 matrix. It represents the resulting homogeneous transform for the specified roll/pitch/yaw angles.

Description

Return a homogeneous transformation for the specified roll/pitch/yaw angles in radians. These correspond to rotations about the Z, Y, X axes respectively.

Examples

   // The following example shows how rt_rpy2tr() can be used to generate
   // a homogeneous transform for specified roll/pitch/yaw angles in a
   // direct approach.

   // roll/pitch/yaw angles
   roll = %pi/6;
   pitch = %pi/4;
   yaw = -2/5*%pi;

   // corresponding homogeneous matrix obtained
   // by compounded transforms
   T1 = rt_rotz(roll) * rt_roty(pitch) * rt_rotx(yaw),

   // by using rt_rpy2tr() (direct approach)
   T2 = rt_rpy2tr(roll, pitch, yaw),
