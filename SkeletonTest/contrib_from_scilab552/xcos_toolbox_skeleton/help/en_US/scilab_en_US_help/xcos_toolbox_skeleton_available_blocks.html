Example page to document some fonctionnalities

Schema

Open the schema

Description

This page describe how to use the Xcos specific helptools provided with Scilab. Using these tools you can automatically generate a diagram view and add a link to open it inside Xcos.

To add a link to an Xcos schema inside an help page, you should use the standard link element with a type="scilab" attribute and a toolbox specific linkend url.

Example for the xcos_toolbox_skeleton toolbox:

...
<link type="scilab" linkend="xcos_toolbox_skeleton.xcos/demos/Blocks.zcos">Open the schema</link>
...

To render an Xcos schema inside an help page, you should use the standard imagedata element with a fileref attribute to an Xcos schema.

Example for the xcos_toolbox_skeleton toolbox and the demos/Blocks.zcos:

...
<inlinemediaobject>
    <imageobject>
        <imagedata align="center" fileref="../../demos/Blocks.zcos" valign="middle"/>
    </imageobject>
</inlinemediaobject>
...
